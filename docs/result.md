# 模型结果
## 第一次训练的模型
- 模型`"experiments\simclr\checkpoints\simclr_epoch400.pth"`
### 5-shot
- 监督基线 5-shot 准确率 = 17.59% 
- 冻结编码器 5-shot 准确率 = 59.88%
    - 变化趋势：一开始变换很快到50%，30epoch后变化减缓
- 全微调 5-shot 准确率 = 49.25%
    - 整体变化速度平稳
### 10-shot
- 监督基线 10-shot 准确率 = 22.63% 
- 冻结编码器 10-shot 准确率 = 68.09%
    - 仍是30epoch后变化减缓，已到达66%
- 全微调 10-shot 准确率 = 63.79%

### 准确率对比

| 方法 | 5-shot | 10-shot |
|------|--------|---------|
| 监督基线（随机初始化） | 17.59% | 22.63% |
| 线性评估（SimCLR 预训练 + 冻结编码器） | 59.88% | 68.09% |
| 全微调（SimCLR 预训练 + 更新全部参数） | 49.25% | 63.79% |

### 观察
- 在 5-shot 下，线性评估远超全微调（59.88% vs 49.25%），说明数据极少时冻结特征更稳健。
- 在 10-shot 下，全微调与线性评估的差距缩小（68.09% vs 63.79%），更多数据让微调开始发挥作用，但仍未超越线性评估。
- 监督基线始终最低，表明预训练带来的特征表示对少样本学习至关重要。
